# Análise de Anúncios de Veículos nos EUA

Aplicação web interativa para análise exploratória de dados de anúncios de veículos usados nos EUA.

## Funcionalidades

- Histograma da distribuição de preços dos veículos
- Gráfico de dispersão entre odômetro e preço, segmentado por condição do veículo
- Interface interativa com checkboxes

## Tecnologias

- Python
- Streamlit
- Pandas
- Plotly Express

## Como executar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```
